# carrot-hunt
A text-based game for teaching Javascript.

Requires [`ionicons`](http://ionicons.com/) for the carrot icon, if you can see the carrot below, you are OK.

## How to run
``` javascript
$ git clone https://github.com/dcousens/carrot-hunt.git
$ cd carrot-hunt
$ node go.js
bunny> print()
```
